# OpenClimb UI Specification

## Philosophy

OpenClimb is designed for climbers.

The core principle of the project is:

> **The map is the heart of the application.**

When the application starts, users immediately see the map and can begin planning their climbing.

---

# Screen 1 – Map

The map is the main screen of the application.

Zooming in reveals the following hierarchy:

Area → Crag → Sector → Route

Users can also open a sector directly by zooming in on the map.

---
