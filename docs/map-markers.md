# OpenClimb Map Markers

## Purpose

Map markers are the primary navigation element in OpenClimb.

Users should be able to understand a climbing area at a glance without opening additional screens.

---

# Marker Design

Each marker is a circle divided proportionally by climbing type.

Current climbing types:

🟢 Sport

🟡 Trad

Future climbing types:

🟣 Drytool

🟤 Boulder

Additional climbing types may be added without changing the marker concept.

---

# Marker Content

The center of the marker displays the total number of climbing routes.

Example:

Total routes: 32

Sport: 24

Trad: 8

The circle should visually represent the percentage of each climbing type.

---

# Zoom Levels

## Country Level

Display only climbing areas.

Example:

● Jura

● Sokoliki

● Rudawy

---

## Area Level

Display proportional markers.

The marker shows:

- total routes
- climbing type distribution

---

## Sector Level

Individual sectors become visible.

Selecting a sector opens the sector page.

---

## Route Level

The user opens a single climbing route.

---

# Design Principles

- Minimalistic
- Easy to understand
- One glance should be enough
- No unnecessary text on the map
- Colors communicate climbing type
- Numbers communicate route count

---

# Future Expansion

The marker system must support additional climbing disciplines without redesigning the interface.
