# Sector Screen

## Purpose

The Sector Screen helps climbers quickly choose a climbing route.

This is the most frequently used screen while climbing.

---

## User Goal

Choose a route with minimal scrolling.

---

## Success

The user finds a suitable route within seconds.

---

# Layout

Header

- Sector name

Hero Image

- Sector photo

Topo

- Interactive topo

Sector Information

- Number of routes
- Difficulty range
- Maximum height
- Sun exposure
- Approach time

Route List

Every route displays:

- Route name
- Official grade
- Climbing type
- Route height
- Number of quickdraws
- Protection character
- Route character
- User rating

Selecting a route opens the Route Screen.
