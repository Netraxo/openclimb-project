# Home Screen

## Purpose

The Home Screen is the main entry point of OpenClimb.

The application always starts on the map.

Users should be able to plan an entire climbing day directly from this screen.

---

# Layout

Top Bar

- Search
- Filter

Main View

- Interactive Map

Floating Button

- Current Location (GPS)

Bottom Navigation

- Map
- Saved
- Profile

---

# Search

The search field allows searching for:

- Area
- Crag
- Sector
- Route

Selecting a result opens the corresponding page.

---

# Filters

Available filters:

Difficulty

Climbing Type

- Sport
- Trad

Aspect

- North
- East
- South
- West

Applying filters updates the map immediately.

---

# Map

The map is the primary navigation system.

Users navigate by zooming.

Navigation Levels

Country

↓

Area

↓

Crag

↓

---

# Markers

Each climbing area is represented by a proportional circular marker.

The marker displays:

- Total number of routes
- Distribution of climbing types

Current climbing types:

🟢 Sport

🟡 Trad

Future climbing types:

🟣 Drytool

🟤 Boulder

# Design Principles

- Map First
- Minimal UI
- One-hand operation
- Maximum two taps to reach any climbing area
- No unnecessary buttons
- The map always remains visible
